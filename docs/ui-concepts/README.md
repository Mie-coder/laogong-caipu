## 主流程

```text
首页
→ 点击“从小红书导入菜谱”
→ 打开导入抽屉
→ 粘贴内容并开始解析
→ 展示四阶段解析进度
→ 进入图片审核
→ 进入菜谱确认
→ 保存并打开菜谱详情
→ 点击“标记做过”
→ 填写并保存复盘
```

## 关键交互

### 首页

- 点击导入行：从底部打开导入抽屉。
- 点击最近做过：进入对应菜谱详情。
- 点击“菜谱”：切换到菜谱列表。

### 导入

- 输入为空时，“开始智能解析”禁用。
- 粘贴成功后保留原始文本。
- 点击遮罩、关闭按钮或下滑：关闭抽屉。
- 解析失败时返回抽屉并保留输入。

### 解析

- 四个阶段按顺序完成，当前阶段使用珊瑚色小范围强调。
- 点击取消：二次确认后返回导入抽屉。
- 解析成功：自动进入图片审核。

### 图片审核

- 点击缩略图：切换选中状态和主图预览。
- 点击“设为封面”：更新封面标识。
- 允许取消全部图片并无图保存。
- 确认后进入菜谱确认页。

### 菜谱确认

- “概览、食材、步骤”使用下划线标签切换。
- 食材和步骤可新增、编辑、删除和排序。
- 菜名为空或步骤为空时禁止保存。
- 离开未保存页面时弹出确认。
- 保存成功后进入菜谱详情。

### 菜谱列表

- 搜索按菜名实时过滤。
- 筛选使用底部抽屉，不增加独立分类导航。
- 点击菜谱进入详情。
- 点击管理进入多选删除状态。

### 菜谱详情

- “备料、步骤”切换时滚动到对应内容。
- 食材可勾选，勾选状态仅用于本次做菜。
- 点击图片进入全屏轮播。
- 点击“标记做过”打开复盘抽屉。

### 做菜复盘

- 至少填写评分或一项文字内容后才能保存。
- 保存后关闭抽屉、做过次数加一，并更新最近复盘。
- 成功反馈使用短暂 Toast，不使用全屏庆祝。

## 视觉约束

- 暖白约 80%，深墨色约 15%，珊瑚色不超过 5%。
- 不使用大面积彩色卡片、胶囊按钮或粗边框。
- 用真实菜品图片、字体层级、留白和发丝线组织页面。
- 圆角控制在 6–8px，底部导航只保留“导入、菜谱”。
